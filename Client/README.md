📰 Indian News Web App

A modern and dynamic Indian news application built with React.js, Firebase, and Tailwind CSS.
This version uses the NewsData.io API to fetch real-time Indian news across multiple categories.
It includes secure user authentication, a comment system, session-based bookmarking, and smooth pagination for browsing articles easily.

This is actually full stack version of my previous project, it had CORS issues, I try to solve that issue with creating a small backend.
